このページでは、MaaS Blender の各コンポーネントと、データフォーマット仕様を含む外部依存関係との間のバージョン互換性をまとめています。
入力データの準備や個別コンポーネントのアップグレードの際にご参照ください。

## GTFS Flex 互換性

[GTFS Flex](https://gtfs.org/extensions/flex/) は、初期のドラフト仕様と GTFS 標準に正式に組み込まれたバージョンとの間で構造的変更がありました。
この2つのバリアントは、ロケーショングループのメンバーシップと停車時刻の表現方法が異なります。

### 仕様の違い

| フィールド / ファイル            | GTFS Flex 仕様ドラフト                        | GTFS Flex 仕様標準                   |
|-------------------------|-----------------------------------------|----------------------------------|
| ロケーショングループの参照           | `location_groups.txt` の `location_id` 列 | 別ファイル `location_group_stops.txt` |
| `stop_times.txt` の停留所参照 | `stop_id` 列                             | `location_group_id` 列            |

### コンポーネントサポートマトリクス

凡例: `⚪ Legacy` = 旧バージョン（メンテナンスのみ）、`🟢 Current` = 安定版/リリース済み、`🛠 Planned` = 予定/進行中。
