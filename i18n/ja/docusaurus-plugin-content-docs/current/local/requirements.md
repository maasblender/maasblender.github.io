## 必要なソフトウェア

以下のソフトウェアを事前にインストールしてください。
- Docker および Docker Compose （あるいはこれらを代替可能なもの）
- Python 3.x
  - 必要な Python パッケージ：`httpx`

## 確認済みの動作環境

| ソフトウェア   | バージョン       |
| -------------- | ------------------ |
| OS             | Ubuntu 22.04.5 LTS |
| Docker         | 28.0.1             |
| Docker Compose | v2.33.1            |
| Python         | 3.10.12            |
| httpx          | 0.28.1             |
