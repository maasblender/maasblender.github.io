以下のソフトウェアがご利用のマシンにインストールされていることを確認してください。

- **Git** — クイックスタートに使用するリポジトリをクローンするために使用します。  
- **Docker** および **Docker Compose** — シミュレーションをコンテナとして実行するために使用します。  
- **Python 3.9 以上** — MaaS Blender の各サービスと連携し、シミュレーションを実行するための実行スクリプトを動かすために使用します。  

## Python 依存関係のインストール

クイックスタートガイドで使用する実行スクリプトは、HTTP リクエストを送信するために `httpx` ライブラリに依存しています。

以下のコマンドで `httpx` ライブラリをインストールしてください。

```shell
pip install httpx
```
