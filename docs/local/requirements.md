## Required Software

Please install the following software in advance:
- Docker and Docker Compose (or compatible alternatives)
- Python 3.x
    - Required Python package: `httpx`

| OS             | Ubuntu 22.04.5 LTS |
| Docker         | 28.0.1             |
| Docker Compose | v2.33.1            |
| Python         | 3.10.12            |
| httpx          | 0.28.1             |
